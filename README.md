# Tenacious Bench

This folder contains the Week 11 Sales Agent Evaluation Bench work for Tenacious.

## Goals

- Build a Tenacious-specific evaluation dataset
- Design a contamination-resistant training/eval pipeline
- Train a small LoRA adapter/judge/process model
- Publish dataset, model card, and blog writeup

## Structure

- `methodology.md` — approach, path choice, rubric, and methodology notes
- `dataset_design.md` — dataset schema, task generation plan, and contamination checks
- `budget_log.md` — cost tracking for API and compute
- `artifacts/` — dataset, model, blog, and publication artifacts
